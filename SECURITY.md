# Security

## Purpose

This document explains how to report suspected security issues in this repository.

## Reporting a Security Issue

If you discover a security issue in this repository, please open a private security report using the repository's security reporting process or contact the repository maintainer through the contact information published in the GitHub profile.

Include enough detail to help reproduce or understand the issue.

## Responsible Disclosure

Please do not publish suspected vulnerabilities publicly before the issue has been reviewed and a responsible disclosure path has been agreed.
